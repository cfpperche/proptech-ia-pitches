# Boas práticas de pitch — síntese de pesquisa

Pesquisa feita em maio/2026 sobre estrutura de pitch / one-pager para investidores e aceleradoras. Este documento é a referência que orienta o formato de `pitches/_template.md`.

## Por que one-pager (e não deck longo)

Para um primeiro contato — que é o caso aqui, levar uma ideia ao Rodrigo —, o **one-pager** vence o deck longo. Pesquisa da NFX aponta ~2,5× mais taxa de resposta para one-pagers em outreach frio. O deck completo vem depois, se houver interesse.

## As 7 seções essenciais

Convergência das fontes pesquisadas (Storydoc, Visme, River, Vestbee, ABStartups, Exame):

1. **Headline + proposta de valor** — uma frase que diz quem atende, o que entrega e por que importa.
2. **Problema** — 1 a 3 dores reais. Não mais que 3. Quantificadas.
3. **Solução** — o valor gerado, não a lista de features.
4. **Mercado** — TAM / SAM / SOM. Grande o suficiente para interessar, conectado à solução.
5. **Modelo de negócio** — como o dinheiro entra.
6. **Time** — credenciais relevantes; constrói confiança de execução.
7. **O ask** — o pedido claro: valor, termos, o que se espera do investidor.

A maioria das fontes adiciona ainda **"Por que agora"** — a janela de mercado/regulatória/tecnológica — e **tração/prova** quando existe.

## Princípios que valem mais que a estrutura

- **Lidere com o problema de negócio, não com a tecnologia.** Use a linguagem do cliente (síndico, administradora, lojista), não a do produto. "IA" não é o pitch — o resultado de negócio é.
- **Quantifique resultados específicos.** "Reduz inadimplência de 13% para X" vence "melhora a cobrança".
- **Concisão é o trabalho.** O bom one-pager diz mais com menos. Cada frase muda o que o leitor sabe.
- **Prova vence promessa.** Qualquer tração — protótipo, piloto, dado proprietário, carta de cliente — pesa mais que adjetivo.
- **"Por que agora" cristaliza o valor.** Sem janela, a ideia parece adiável.

## Adaptação ao contexto Startup Academy / Group

- O Rodrigo já sinalizou o critério de triagem dele num comentário público: pediu "um resumo do que você já tem em **estágio de monetização**". Pitch que mostra caminho de receita — ou um MVP — supera ideia solta.
- O ask não deve mirar só o cheque (até R$ 200 mil sem equity é pouco). O prêmio real é o **ecossistema**: base de ~100 mil condomínios, integração nativa, canal de distribuição. O ask de cada pitch reflete isso.
- Cada pitch precisa responder, implicitamente, "por que isto não compete com o que a Group já vende?". Daí a seção *Encaixe no ecossistema Group*.
